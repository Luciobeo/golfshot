# Andrea Golf Caddie

Pacchetto pronto per GitHub Pages.

## Pubblicazione
1. Apri il repository `golfshot`.
2. Elimina o sostituisci i vecchi file nella root.
3. Carica `index.html`, `manifest.webmanifest`, `sw.js`, `icon-192.png`, `icon-512.png` e la cartella `assets`.
4. Mantieni esattamente nomi e struttura.
5. Attendi 1-3 minuti e ricarica la pagina.
6. Su iPhone, se compare ancora la vecchia versione, elimina l'icona dalla Home e aggiungila nuovamente da Safari.

URL previsto: https://luciobeo.github.io/golfshot/
